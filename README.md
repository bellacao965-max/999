Mumet - Sumber Inspirasi (Static site, Green Fresh theme)

Deployment (Render - Static Site):
1. Push repository to GitHub.
2. On Render: New -> Static Site.
3. Connect repo, set Publish directory to 'public', leave Build command empty.
4. Deploy.

Optional: To enable real AI, deploy the optional server (server/server.js) as a Web Service and set PROXY_URL in public/config.js.
